html
<html> represents the root of the HTML document
<head> represents a collection of metadata for the document
<body> element defines the document body
lang is added to specify the document's language for all the text on the page
<h1> element defines a heading, h1-h6
<p> element defines a paragraph
<br> an empty element without a closing tag
<a> defines a hyperlink
href the most important attribute of the <a> element; indicates the link's destination
<hr> element used to separate contentin an HTML page
<title> defines the title of the document
id attribute specifies a unique id for an HTML element
class attribute is used to define equal styles for elements with the same class name
<div> defines a section in an HTML document; it is often used with CSS to layout a web page
<nav> defines a set of navigation links
style specifies an inline style for an element
<canvas> it is used to draw graphics via scripting




css
id selector uses the id attribute of an html element to select a specific element (#id)
class selector selects HTML elements with a specific class atribute (.class)
* it is universal selector; selects all HTML elements on the page
element selects all elements (eg. p>>selects all <p> elements)
